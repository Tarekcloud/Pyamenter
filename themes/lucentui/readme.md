# 🌟 Lucent – Minimalist Theme for Paymenter

Lucent is a clean, modern theme designed for [Paymenter](https://paymenter.org), featuring a glassy interface, smooth transitions, and full responsiveness across devices.

> Designed with 💙 by **Zaqua**

---

## 📦 About

- 🎨 Minimalist glass UI with soft animations  
- 🌗 Dark and light mode support 
- 🛠️ Easily customizable via settings panel
> **Note:** The following features are temporarily deprecated:
>
> - 🚀 Supports PTB `.paytheme` installation
