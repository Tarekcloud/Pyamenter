Lucent for Paymenter

  • 🎨 Minimalist glass UI with soft animations

📥 Installation Guide

1

Upload the ZIP file

Move the downloaded Lucent.zip into your Paymenter themes directory.

root@server:~# mv Lucent.zip /var/www/paymenter/themes

root@server:~# cd /var/www/paymenter/themes

2

Extract the theme

Unzip the archive and ensure the folder structure is correct.

root@server:~# unzip Lucent.zip
/themes └── Lucent/ ├── dashboard.blade.php ├── home.blade.php └── other views...
3

Setup Flag Assets

Locate the flags folder. Move this folder to the public/assets/ directory.

public/ └── assets/ └── flags/ ├── language-en.svg ├── language-id.svg └── ...
4

Build Assets Required

⚠️ Skipping this step may result in a 500 Internal Server Error.

root@server:~# cd /var/www/paymenter

root@server:~# npm i

root@server:~# npm run build Lucent

5

Activate Lucent

Go to your Admin Panel:
Settings → Theme → Choose Lucent.

🖼️ Checkout Icons Setup

You can display custom icons (e.g., Operating System logos) for configurable options during checkout.

1. Directory Setup

Create the following directory and upload your icon images there:

paymenter/public/assets/lucentui/icons/

2. Configuration

Create or edit the config.json file in the directory above. You can map Option IDs to filenames, or use keywords as a fallback.

config.json
{
    // Priority: Match Option ID to Image
    "ids": {
        "3": "ubuntu.png", 
        "4": "windows.png"
    },
    
    // Fallback: Match text in option name
    "keywords": {
        "ubuntu": "ubuntu.svg",
        "centos": "centos.png",
        "game": "https://cdn-icons-png.flaticon.com/512/808/808439.png"
    }
}

*Tip: You can use local filenames (must be in the icons folder) or direct URLs.
